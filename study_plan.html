How to Use the Resources

Daily Rhythm

  • Pre-rounds (20 min): Anki reviews only -- no new cards
  • Lunch (20 min): New cards (NSGY Boards) + 10 new Fluoro cards
  • Evening (25 min): Finish new card target for the day
  • After cases: Search Anki for the case topic tag -- drill 10 cards

Weekly Rhythm (Sunday)

  • 15 min -- Greenberg prep: Open AccessSurgery to next week's chapter. Skim headings, memorize 3 key numbers (doses, grades, thresholds)
  • Set Anki filter to next week's subdeck in NSGY Boards
  • 20 SANS questions: Any topic, timed, read all explanations carefully

Case-Linked

  • Night before cranial case: Open Rhoton atlas for that anatomy (5 min)
  • Night before spine case: Watch Operative Neurosurgery video for that approach
  • After tumor case: Pull WHO classification table + run Anki tumor tag
  • Week 16: Full timed SANS mock exam -- simulate real exam conditions

Anki Auto-Sync (no plugin required)

Run python3 ~/anki_sync.py in Terminal after each study session. It reads your Anki database directly -- Anki does not need to be open. It counts all cards reviewed today from NSGY Boards 2024 Updated, Dura Deck v2::TR Neurosurgery Board Review, and NSGY Fluoro, then opens this page with the count auto-applied. No manual entry needed.
